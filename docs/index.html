📖 API 文档


            

                
🚀 Quick Start


POST /api/generate

curl -X POST http://localhost:8080/api/generate \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "一个穿红色斗篷的精灵猎人",
    "style": "pixel",
    "render_mode": "ai"
  }'

            


            

                
📝 Request Body


{
  "prompt":     "string (required)",   // 角色描述
  "style":      "pixel|cartoon|anime|western|dark",
  "char_type":  "warrior|mage|archer|rogue|healer|monster|npc",
  "seed":       12345,                 // 固定种子可复现
  "render_mode":"procedural|ai|hybrid", // 渲染模式
  "ai_backend": "pollinations",        // AI后端
  "formats":    ["spine","unity","godot"],
  "include_png": true
}

            


            

                
🎨 渲染模式

                

                    

                        模式
                        说明
                        响应时间
                    

                    
procedural程序化生成（默认）<1s

                    
aiAI图像生成 (Pollinations/Flux)~30-60s

                    
hybridAI参考图 + 程序化精灵表~30-60s

                

            


            

                
🔗 所有端点


POST /api/generate   生成角色资产包
GET  /api/info       API信息
GET  /api/styles     可用风格列表
GET  /api/types      可用角色类型
GET  /api/backends   AI图像生成后端
GET  /preview/{id}   预览已生成角色
