# ArtPipe v0.3 — AI图像生成 + 渲染质量飞跃

> **版本:** 0.3.0 ~ 0.3.77 | **Commits:** `1ccd7ce`..`HEAD` (77次迭代) | **日期:** 2025-04 ~ 2026-04

## 里程碑

v0.3 是 ArtPipe 质量飞跃的版本。新增 AI 图像生成集成，并通过 55 次持续迭代将程序化渲染质量从"可辨识"提升到"专业级像素美术"水准。

## 核心新功能

### 🎨 AI 图像生成
- Pollinations.ai 免费后端集成
- HuggingFace FLUX.1-schnell 免费后端
- 3 种渲染模式：`procedural` / `ai` / `hybrid`
- 动态模型选择 + 风格定制负面提示词

### 🌐 GitHub Pages 交互式生成器
- 在线角色生成器预览页
- 实时 API 调用演示
