# ArtPipe v0.2 — API-First 重构

> **版本:** 0.2.0 | **Commits:** `66650f3`..`c432b61` | **日期:** 2025-03

## 里程碑

从 CLI 工具转型为 HTTP API 服务。一个 POST 请求即可获得可直接导入引擎的完整角色资产包。

## 新增功能

### HTTP API
- `POST /api/generate` — 核心接口，提示词 → 完整角色资产包
- `GET /` — 健康检查
- `HEAD /` — HEAD 支持
- JSON 请求/响应格式

### 风格 × 类型矩阵
- **5 种风格：** pixel, cartoon, anime, western, dark
- **7 种角色类型：** warrior, mage, archer, rogue, healer, monster, npc
- 共 35 种风格×类型组合

### 动画系统（扩展）
- 6 种动画状态：idle, walk, run, attack, hurt, die
- 逐帧精灵表生成

### 多格式导出
- **SpriteSheet PNG** — base64 编码
- **Spine 骨骼 JSON** — 可导入 Spine2D
- **Unity Package** — 可直接导入 Unity
- **Godot Scene** — 可直接导入 Godot

## API 示例

```bash
curl -X POST http://localhost:8080/api/generate \
  -H "Content-Type: application/json" \
  -d '{"prompt": "一个穿红色斗篷的精灵猎人，手持弓箭"}'
```

## 架构改进

- 从 CLI 脚本重构为 HTTP 服务架构
- 模块化拆分：`app.py`(服务) + `engine.py`(渲染) + `exporter.py`(导出)
- 零依赖原则保持 — 纯标准库 `http.server`
