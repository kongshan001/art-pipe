# ArtPipe v0.1 — MVP

> **版本:** 0.1.0 | **Commit:** `f0b89d4` | **日期:** 2025-03

## 里程碑

ArtPipe 首个可用版本 — AI 2D游戏角色资产生成管线。

从零构建了程序化角色渲染的核心架构，实现"文字描述 → 像素角色"的基本管线。

## 新增功能

### 渲染引擎
- 基础程序化角色渲染引擎
- 分层渲染架构：头部、身体、四肢、装备独立绘制
- 基础色彩系统，支持随机调色板生成

### 动画系统
- 基础动画状态：idle（待机）、walk（行走）、attack（攻击）
- SpriteSheet 帧动画导出

### 输出
- PNG SpriteSheet 输出
- 命令行接口

## 架构决策

- **纯 Python 标准库** — 零外部依赖，降低使用门槛
- **程序化优先** — 毫秒级生成，不依赖外部 API
- **分层设计** — 为后续 v0.2 的 API 化和 v0.3 的 AI 集成奠定基础

## 文件结构

```
art-pipe/
├── app.py              # 入口
├── engine.py           # 渲染引擎
├── exporter.py         # SpriteSheet 导出
└── artpipe/
    └── __init__.py     # 版本定义
```

## 已知限制（在后续版本中解决）

- 仅命令行接口，无 API
- 动画种类有限（3种）
- 无 AI 图像生成
- 无游戏引擎格式导出
